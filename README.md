## Usage

Install package:

```sh
# npm
npm install ohmygql

# yarn
yarn install ohmygql

# pnpm
pnpm install ohmygql
```

Import:

```js
// ESM
import { GqlClient } from 'ohmygql'

// CommonJS
const { GqlClient } = require('ohmygql')
```
